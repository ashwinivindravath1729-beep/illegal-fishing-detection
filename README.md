# 🌊 Illegal Fishing Detection Using Satellite AIS Data

## 📌 Project Overview
This project analyzes real AIS satellite tracking data to detect 
illegal and unreported fishing activity using Machine Learning.

## 🎯 Key Results
- 📊 Analyzed **1999 real vessel records**
- 🚨 Detected **397 suspicious/illegal vessels**
- 🗺️ Built interactive ocean map
- 🤖 Used Isolation Forest AI model

## 🛠️ Technologies Used
- Python, Pandas, NumPy
- Scikit-learn (Isolation Forest)
- Folium (Interactive Maps)
- Matplotlib, Seaborn

## 📂 Files
- `fishing_analysis.ipynb` - Main project notebook
- `fishing_analysis_charts.png` - Analysis charts
- `illegal_fishing_map.html` - Interactive map
- `sample_ais.csv` - Dataset

## 🚨 Suspicious Behaviors Detected
- High speed anomalies
- Hidden vessel heading
- Missing IMO registration
- Stopped vessels in open ocean
